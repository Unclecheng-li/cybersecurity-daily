Microsoft 7 月 Patch Tuesday：SPNEGO NEGOEX CVE-2025-47981 CVSS 9.8 蠕虫级 RCE，Kerberos RC4 永久退役，三重风暴交汇 7 月 14 日

127 个 CVE 修复、SPNEGO 蠕虫无需凭据无需交互、NightmareEclipse 7·14 D-Day 同日到来——Windows 运维团队面临全年最紧张的一天。

1 严重

CVE-2025-47981 — SPNEGO NEGOEX 堆缓冲区溢出 CVSS 9.8 蠕虫级 RCE，所有 Windows 默认受影响

Microsoft MSRC / Sophos / Rapid7 / ZDI 7/14 发布 CVSS 9.8 蠕虫级

微软 7 月 Patch Tuesday 共修复 127 个 CVE（9 个 Critical、118 个 Important），其中 34 个 CVSS 8.0+。最严重的是 CVE-2025-47981——SPNEGO Extended Negotiation（NEGOEX）安全机制中的堆缓冲区溢出（CWE-122）。SPNEGO 位于 Kerberos 和 NTLM 之下，是认证协商的底层协议，默认启用于所有 Windows 10 1607+ 和 Windows Server 2008R2+。

技术参数：

属性值
CVSS v3.19.8 (AV:N/AC:L/PR:N/UI:N/S:U/C:H/I:H/A:H)
攻击向量网络 — 无需凭据、无需用户交互
蠕虫传播✅ 可自动扫描并感染内网其他主机
Microsoft 评估30 天内极可能被利用

Rapid7：预认证漏洞，攻击者可借此获取特殊权限，微软评估短时间内即出现野外利用。ZDI：利用无需用户交互，具备蠕虫特质——在 Patch Tuesday 历史中极为罕见。

⚠️ Kerberos RC4 永久退役：7 月 14 日累积更新同时永久移除 RC4 回滚机制。任何仍使用 RC4 的服务账户打补丁后认证失败——先审计再打补丁。

攻击者发送恶意 NEGOEX 消息 堆缓冲区溢出触发 SYSTEM 权限 RCE 蠕虫自动传播内网 域控制器沦陷
2 严重

NightmareEclipse 7·14 D-Day 到来 — 7 个 Windows 零日后的"粉碎骨头"威胁日与 Patch Tuesday 交汇

byteiota / TechTimes / The CyberSignal 7/14 3 个在野利用 3 个未修复

自 4 月起连续披露 7 个 Windows 零日的 NightmareEclipse（Chaotic Eclipse），其5 月 28 日威胁的"bone-shattering"发布日——7 月 14 日——正式到来。目前 BlueHammer、RedSun、UnDefend、RoguePlanet 已修复；YellowKey（60 秒绕过 BitLocker）、GreenPlasma、MiniPlasma 仍无补丁。

三重风暴交汇：Patch Tuesday 127 CVE + Kerberos RC4 强制退役 + NightmareEclipse D-Day。Windows 团队面临全年最高压运维窗口。

研究员因 RoguePlanet 消耗精力"部分收回"威胁，但未正式取消。社区正密切关注其自托管代码平台动向。

3 严重

Jamf 披露 CrashStealer — 原生 C++ macOS 窃密器，绕过 Gatekeeper 针对 80 个加密钱包 + 14 款密码管理器

Jamf Threat Labs 7/14 报告 / AppleInsider 在野利用 AES-256-GCM

Jamf 7 月 14 日正式发布 CrashStealer 完整分析报告——全 C++ 编写的 macOS 窃密器，通过合法 Apple 公证+Developer ID 签名的 Werkbit 磁盘镜像分发，可完全绕过 Gatekeeper。

目标覆盖：

类别目标数
加密钱包扩展~80 个
密码管理器14 款
浏览器6 款
系统数据钥匙串+文件

采用控制流平坦化+反调试+AES-256-GCM 加密+LaunchAgent 持久化——macOS 恶意软件的专业化里程碑。

4 严重

Gitea act_runner CVE-2026-58053 — CVSS 9.9 容器逃逸，PoC 公开，全球 11.3 万风险资产

exploitarium / VulnCheck / 奇安信 CERT CVSS 9.9 官方无补丁

Gitea act_runner Docker 后端（act ≤0.262.0）在配置 privileged: false 时仅关闭 Privileged 标志，但 container.options 中的 --pid=host、--cap-add=SYS_ADMIN 等危险选项原样合并到 Docker HostConfig。

全球影响力：奇安信测绘 113,706 个风险资产，40,732 个关联 IP。官方暂无补丁，临时缓解：限 workflow 权限 + 外部 PR 手动审批 + Runner 网络隔离。

攻击者只需提交恶意 workflow 即可逃逸至宿主机 root——一旦 Runner 沦陷，所有仓库密钥全部暴露。

5 高危

Lidl 欧洲三国供应链数据泄露 + 巨騰國際暗网客户数据曝光

网易 / AASTOCKS / Lidl 官方声明 7/14 供应链攻击

两起供应链数据泄露同日曝光：Lidl 第三方 IT 服务商遭攻击，荷兰/比利时/德国在线商城顾客全名+电话+邮箱+出生日期被盗，公司罕见直白警告用户防范钓鱼；巨騰國際（港股 03336）暗网发现客户/供应商数据被出售，已紧急聘请网络安全顾问。

两起事件共同指向："企业自身安全≠供应链安全"——第三方服务商正成为最薄弱环节。

速览区

6. Adobe 宣布 Patch Tuesday 翻倍至每月两次 — AI 加速漏洞发现推动补丁节奏变革

Adobe 从 7 月起安全公告增至每月两次（第二和第四个周二），直接原因是攻击者利用 AI 加速漏洞发现。InfoSec Industry:"利用窗口以小时计，月度补丁周期在数学上不再成立。"

7. DuneSlide Cursor AI IDE 双零点击 RCE（CVE-2026-50548/50549）CVSS 9.8

Cato AI Labs 发现 Cursor IDE 中两个 CVSS 9.8 零点击 RCE。攻击者通过 Prompt 注入操纵 AI 代理的工作目录参数或符号链接规范化绕过，实现完全沙箱逃逸。Tata Communications 7/14 列为本周首要威胁。

8. js-yaml CVE-2026-59870 — PoC 公开，O(n²) DoS 可自动化利用，月下载超 1 亿次

js-yaml 5.0.0-5.2.0 YAML11_SCHEMA !!omap 标签 O(n²) CPU 消耗导致 DoS。CISA ADP 标记"exploitation: poc"+"automatable: yes"。月下载量超 1 亿次，影响面极广。立即升级至 5.2.1。

9. EU/UK 联合制裁俄罗斯 FSB Center 16 — 12 国联合公告确认全球政府路由器扫描

EU/UK 7/13 制裁 24+ 名俄罗斯 FSB Center 16 成员，正式归因 2025 年 12 月波兰电网攻击。12 国联盟公告确认 FSB 正全球扫描政府路由器弱 SNMP（CVE-2018-0171）。APT28 同步生成 4 个新恶意样本。

10. Google Chrome CVE-2026-15114 — Codecs 越界读写，补丁已发布

Chrome 150.0.7871.115 修复 Codecs 组件越界读写漏洞（CVE-2026-15114），攻击者可通过特制视频文件触发堆损坏 RCE。Chromium 安全等级：High。立即更新 Chrome。
