mySites.guru 披露：iCagenda CVE-2026-48939 零日利用时间线 — 6 月 15 日起自动化攻击，28 天后才入 KEV

mySites.guru / The Hacker News 7/13 公开 / CISA KEV 7/10 加入 CVSS 10.0 KEV 零日确认

The Hacker News 7 月 13 日报道，WordPress/Joomla 托管安全服务商 mySites.guru 证实CVE-2026-48939（iCagenda 未认证任意文件上传 RCE）自 2026 年 6 月 15 日起即被以零日方式在野利用，攻击者通过 POST index.php?option=com_icagenda&task=registration.submit 上传 PHP webshell。CISA 直到 7 月 10 日才加入 KEV——从首次利用到官方确认，整整 28 天防守真空期。

2 严重

Joomla RSFiles CVE-2026-57827 — 本周第五个 CVSS 10.0 满分未认证 RCE

Enigma Global Daily Digest 7/13 / NVD CVSS 10.0

Joomla 扩展 RSFiles（文件管理与下载组件）被发现存在未认证任意文件上传漏洞，CVSS 10.0 满分。攻击者无需登录即可上传恶意文件至公开目录，直接获取 RCE。所有未修补版本均受影响。

本周 Joomla 五连洞——iCagenda + Balbooa Forms + SP Page Builder + Page Builder CK + RSFiles，全部未认证文件上传 RCE，攻击模式完全一致。攻击者可能已建立统一扫描→利用→维持工具链。

3 严重

Flowise CVE-2026-56271 — 硬编码 JWT 密钥 CVSS 9.3，可伪造管理员令牌窃取 LLM 凭据

Enigma Global Daily Digest 7/13 / NVD CVSS 9.3 认证绕过

开源 LLM 低代码平台 Flowise（≤3.0.13）硬编码 JWT 签名密钥——auth_token 和 refresh_token 为固定值。攻击者获知后即可伪造任意用户（含管理员）的 JWT，获取所有存储的 LLM 提供商 API 密钥（OpenAI/Anthropic/Gemini/Azure 等）。

Flowise GitHub 60K+ stars，企业部署量大。加上 Langflow CVE-2026-55255 IDOR 形成 AI 中间件漏洞链——全栈 LLM 凭据沦陷。

修复：立即升级至 3.0.14+，轮换所有存储 API 密钥。

4 严重

Ubiquiti UniFi OS SAB-066 — CVSS 10.0 + 6 Critical + 1 High，网络可达即完全接管

Ubiquiti SAB-066 / Enigma Global 7/13 10.0 + 6×Critical

Ubiquiti 通过安全公告 SAB-066 披露 UniFi OS 设备集群漏洞，共 8 个 CVE（1 个 CVSS 10.0、6 个 Critical、1 个 High），覆盖命令注入、认证绕过、未授权 API 访问。

加上 7/8 的 11 个 UniFi CVE，UniFi 生态两周内已累积 19+ 严重漏洞。Shodan 可检索数十万暴露 UniFi 设备，MSP 托管场景中一台沦陷=数十家客户连带暴露。

5 高危

Defender RoguePlanet CVE-2026-50656 — 补丁引入新 Bug：SpyNet 数据泄露 + 磁盘耗尽

NightmareEclipse 7/13 公开 / 网易 补丁回归 Bug

微软 7/9 紧急修复 Defender RoguePlanet 提权漏洞后，补丁自身引入回归问题：

  • 8 字节数据泄露：mpengine.dll 在特定文件打开操作中泄露数据
  • SpyNet 武器化：结合 SpyNet 自动样本提交，攻击者诱使 Defender 反复隔离大文件，耗尽系统卷全部空间
  • 目前无修复——微软尚未回应补充披露

速览区

6. RabbitMQ 11 CVE 集群最高 CVSS 9.2 — 消息队列基础设施全面沦陷

RabbitMQ 一次性披露 11 个安全漏洞（最高 CVSS 9.2），攻击者可完全接管消息队列中间件——读取/篡改/重放所有异步消息、窃取连接凭据、横向移动至消费者与生产者。RabbitMQ 广泛部署于金融交易、电商订单、微服务通信等关键业务链路。

7. Crawl4AI 三连 Critical CVE — 未认证端点窃取 LLM API 密钥

CVE-2026-56259（CVSS 8.8）、CVE-2026-56260（CVSS 9.1）、CVE-2026-56261（SSRF）覆盖 Crawl4AI Docker API 0.8.8 前版本。未认证 API 端点可重定向 LLM 调用目标、读取环境变量中 API 密钥、发起 SSRF 攻击内部服务。

8. Ghostcommit — 在 PNG commit 中隐藏 Prompt 注入攻击 AI 代码审查

新型攻击技术：攻击者在 Git commit PNG 截图中嵌入隐形恶意提示文本，当 AI 代码审查工具（Claude/Cursor/GitHub Copilot 等）分析 commit 时被注入恶意指令，窃取仓库密钥或篡改代码建议。人类审查者完全无感知。

9. 俄罗斯入侵北约成员国民用 IP 摄像头建立实时监控网络

kyber.today 7/13 报道：俄罗斯威胁组织入侵北约国家民用 IP 摄像头（含门铃），建立实时军事运输监控网络，监视向乌克兰转运武器装备路线。民用 IoT 设备成为国家级地理情报刺探入口。

10. Operation Muck and Load — 222 个 GitHub 仓库投放恶意 Go 模块

Enigma 7/13 报道：大规模 Go 模块供应链攻击，波及 222 个 GitHub 仓库。恶意 Go 模块在开发者执行 go get 时植入后门，攻击者批量注册与合法包名相近的恶意模块（typosquatting）。

11. D1R 勒索利用半导体供应链单一漏洞攻击 Synopsys/Arm/Bosch

kyber.today 7/13 报道：新兴勒索组织 D1R 利用半导体行业单一供应链漏洞同时攻陷 Synopsys（EDA 工具）、Arm（芯片 IP）和 Bosch（汽车电子），体现"一次漏洞、行业级影响"的供应链攻击威力。

12. Comfast CF-WR631AX 路由器 CVE-2026-15511 CVSS 8.9 命令注入

NVD 7/12 收录：Comfast CF-WR631AX 路由器固件至 2.7.0.8 在 FastCGI 后端存在命令注入，攻击者可获取路由器完全控制权，用于 DNS 劫持、内网横移。与 RedHook Android RAT 形成"路由+移动"联合攻击面。

13. Progress ShareFile SZC 紧急关停持续 — 仍无补丁

Progress Software 对 ShareFile Storage Zone Controller 的紧急关停令仍在生效，未公布漏洞详情也未发布补丁。7/10 至今所有本地 SZC 服务器应保持关闭，已关闭客户须排查 CVE-2026-2699/2701 历史利用痕迹。
