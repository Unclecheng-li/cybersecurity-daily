1 严重

Gogs CVE-2026-52813 — 路径遍历 → Git hooks 覆写 → 服务端 RCE

CVE-2026-52813 / GHSA-c39w-43gm-34h5 | CVSS 9.8 PoC已公开

自托管 Git 服务 Gogs 曝出 2026 年最严重的代码托管平台漏洞之一。组织名称字段未对路径遍历序列（../）做严格校验，攻击者通过创建嵌套仓库结构，可以覆写其它仓库的 Git hooks 配置，结合 Gogs 内部仓库缓存机制实现远程代码执行。

漏洞机制：

  • 类型：CWE-23 路径遍历 → 远程代码执行
  • 攻击方式：注册恶意组织名(含 ../) → 嵌套仓库覆写目标 hooks → 触发 git rebase 执行恶意代码
  • 影响范围：所有使用 Gogs < 0.14.3 的自建 Git 服务（开发团队/企业内网/CI/CD 基础设施）
  • 后果：获得 Gogs 服务进程权限，可访问所有托管仓库源代码、SSH 密钥、用户凭证

值得关注的原因：

  • PoC 已公开且复现条件简单，GitHub Advisory、GitLab Advisories Database、Ionix、DailyCVE 多家平台同步预警
  • 与近期 Klue 供应链攻击、Nx Console 投毒形成"开发工具链连环打击"趋势
  • GitHub Advisory 标记为 Critical，GitLab Advisories 同步收录
  • 建议所有 Gogs 实例立即升级至 0.14.3 或更高版本
注册恶意组织名 创建嵌套仓库 覆写Git hooks 触发rebase RCE执行

3 高危

Squidbleed CVE-2026-47729 — 潜伏 29 年的 Squid 堆缓冲区越界读取

CVE-2026-47729 | CVSS 7.5 AI辅助发现

Squid 代理服务器曝出潜伏 29 年的"长寿漏洞"。FTP LIST 目录解析逻辑处理字符串结束符时，对 NULL 终止与 strchr 边界条件判断不一致，导致堆缓冲区越界读取，泄露明文账号、Cookie、会话凭证。

发现方式：AI 安全厂商 Calif 使用自研 AI 代码审计模型辅助识别。PoC 已公开，腾讯云、启明星辰、奇安信等多家国内安全厂商同步发布预警。

4 严重

CISA 紧急 Cisco UCM 漏洞限期修复 — 周日截止

CVSS 9.8 Critical | 在野利用 KEV限期

CISA 于 6 月 26 日发布紧急指令，要求所有联邦民事机构在周日（6 月 28 日）前修复 Cisco Unified Communications Manager 严重漏洞。该漏洞位于 Web 组件中，允许未授权攻击者通过 SSRF 进行任意文件写入，并可进一步提权至 Root。

CUCM 是企业/政府电话系统核心，沦陷后通话录音、SIP 信令、用户凭证全部泄露。

CVSS 9.8 | 已逾期 1 天 KEV截止

Splunk Enterprise 严重漏洞 CVE-2026-20253 的 CISA KEV 修复截止日为 6 月 26 日，今日是截止后第一天。漏洞位于 PostgreSQL Sidecar 端点，由于缺乏身份验证，允许未授权远程创建/截断任意文件，攻击者可进一步组合实现 RCE。
