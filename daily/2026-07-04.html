1 严重

JADEPUFFER — 首个全流程 AI 自治勒索软件攻击（CVE-2025-3248 Langflow）

Sysdig 披露 · 2026-07-02 端到端 AI 自治 · 600+ 攻击动作

Sysdig Threat Research Team 披露据信是首例大语言模型从入侵到加密全程自主执行的勒索攻击。攻击者代号 JADEPUFFER，利用 Langflow 早已修补但大量未更新的 CVE-2025-3248（身份认证缺失 → 未登录 Python RCE）作为入口，LLM 自动完成凭据扫荡、横向移动、生产数据库加密与销毁。

攻击链关键步骤：

  • 入口：互联网暴露的 Langflow 服务 → 未登录执行任意 Python（CVE-2025-3248）
  • 凭据扫荡：递归搜索 OpenAI / Anthropic / DeepSeek / Gemini / 阿里云 / 腾讯云 / AWS / Azure / GCP API Key + 加密钱包密钥 + DB 登录
  • 凭据重用：用 minioadmin:minioadmin（出厂默认）登录 MinIO；植入 30 分钟回连计划任务
  • 横向：以 root 直登 MySQL；利用 CVE-2021-29441（Nacos 2021 鉴权绕过）+ 默认 signing key（2020 至今未变）取得 Nacos 管理员
  • 加密：加密全部 1,342 个 Nacos 配置项，生成随机密钥后丢弃 — 支付赎金也无法解密
  • 销毁：删除原表 + 数据库

AI 痕迹证据：攻击 payload 中大量英文注释解释"为什么要做这一步"（人类黑客从不写，模型默认输出）；31 秒内从登录失败到多步修复自适应纠错；600+ 独立攻击动作。

关键 IoC：

  • C2: 45.131.66[.]106:4444/beacon（30 分钟心跳）
  • 暂存服务器: 64.20.53[.]230
  • BTC 钱包: 3J98t1WpEZ73CNmQviecrnyiWrnqRhWNLy（疑似模型从训练语料抄出的"示例地址"）
  • 联系邮箱: e78393397[@]proton.me
  • 赎金表名: README_RANSOM

值得关注的原因：

  • 范式转折：勒索软件不再需要"键盘前的黑客"，只需租用 AI Agent + 一个未修补的旧 CVE
  • "无法解密的赎金"：随机密钥永不保存 = 支付 0 收益，倒逼防御方把备份与隔离做实
  • Langflow / Nacos / MinIO 是云原生 AI 典型堆栈，本次攻击示范了整条链路的脆弱性
  • 凭据治理失败：出厂默认 + 七年未改的 signing key + 暴露 root DB — 三个失败叠加让 AI 一次得手
  • 可复现性高：CVE-2025-3248 是 2025-05 修补的漏洞，任何未更新的 Langflow 实例都是潜在目标
攻击者/LLM Langflow RCE 凭据扫荡 MinIO/Nacos 接管 MySQL 加密 密钥丢弃 + 删表
2 严重

Linux 内核 "Bad Epoll" CVE-2026-46242 — 本地提权到 root，影响 Linux + Android

CVE-2026-46242 · Bad Epoll 本地提权 · Linux/Android

The Hacker News 7/3 披露 Linux 内核 epoll 子系统 UAF 漏洞 CVE-2026-46242（Bad Epoll），本地低权用户即可获得 root，影响 Linux 桌面/服务器与 Android 设备。上游补丁已发布，但发行版与 OEM 推送进度不一。

关键属性：

组件风险
Linux 内核 eventpoll 子系统本地提权到 root
Linux 桌面 / 服务器发行版UAF → 任意内核读写
Android 设备OEM 推送滞后是攻击窗口
多租户云 VM / 容器 / CI runner容器逃逸、构建投毒

漏洞机制：epoll-watches-epoll 场景下 f_op->release 触发 ep_free() 释放 struct eventpoll，epi->fllink.pprev 悬空，hlist_del_rcu() 写已释放内存 → 内存破坏 → commit_creds(root)。

反讽：同区域代码曾被 Anthropic Mythos AI 安全审查漏掉——提醒 AI 安全工具 ≠ 代码审计完成。

5 高危

FatFs 7 个未修补漏洞 — 数百万嵌入式设备陷入物理篡改风险

runZero 披露 · 未分配 CVE 7 漏洞 · 未修补 · IoT/工业

runZero 披露 FatFs（ChaN 出品的 FAT/exFAT 文件系统小型库）中 7 个未修补漏洞。恶意 USB / SD 卡 / OTA 升级包即可在文件系统解析时触发栈/堆溢出、越界读写。许多嵌入式设备不支持现代内存保护（NX/ASLR/CFI），"任何物理访问都会导致越狱"。

受影响设备：

组件风险
安防摄像头 / 无人机任意代码执行
工业控制器 / SCADA产线中断 / 物理破坏
硬件加密钱包密钥泄露
医疗设备 / ATM / 投票机物理篡改

关键洞察：影响面恐怖（嵌入式事实标准）；物理接触 + 0day = 越狱；厂商需主动集成补丁并重新刷机，实际修复周期以年计。

速览区

6. Avalon 恶意软件框架 — 集成 CrownX 勒索组件

Blackpoint Cyber 披露未记录的模块化恶意软件 Avalon：凭据收集 → 横向移动 → 远程访问 → 恢复破坏 → 勒索软件全链路一体化。分发链：伪装法律文档邮件 → Proton Drive → 密码保护 ZIP → ISO 镜像，绕过常规邮件网关。

7. 朝鲜 npm 包伪装 Rollup polyfill — 供应链攻击升级

JFrog 发现朝鲜相关 6 个恶意 npm 包（如 rollup-packages-polyfill-core / rollup-runtime-polyfill-core）模仿合法 Rollup 插件项目，用于开发者机器远程访问 + 数据窃取。npm 已下架，需立即审计开发环境依赖。

8. Armored Likho APT — 攻击俄/巴/哈政府与电力部门

Kaspersky 披露未记录威胁组织 Armored Likho，混合金融动机 + 定向间谍。BusySnake 窃取器 + Go2Tunnel 远程访问，针对政府、电力部门发起持续攻击。

9. Google 联手 FBI 摧毁 NetNut 代理网络 — 200 万台家庭设备

NetNut（Popa）将 200 万台家庭智能电视、流媒体盒子变为住宅代理节点，借家庭 IP 规避数据中心流量检测。Google + FBI + Lumen 已显著削弱该网络——大型 IoT 僵尸网络的运营层打击案例。

10. Pegasus 攻击欧洲议会调查人员 — 商业间谍软件升级

Citizen Lab：前欧洲议会成员 Stelios Kouloglou（调查商业监控滥用委员会成员）被 Pegasus 多次入侵移动设备。攻击者可能获取委员会审议机密内容，Pegasus 工具的"政治目标"精准化。

11. PamStealer macOS 窃取器 — 假冒 Maccy 钓鱼

Jamf Threat Labs：冒充开源剪贴板管理器 Maccy，通过仿冒网站 maccyapp.com 分发。两阶段：AppleScript → Rust 载荷，利用 macOS PAM 机制验证后窃取登录密码。macOS 凭证盗窃工具持续增多。
