SimpleHelp CVE-2026-48558 满分漏洞遭在野利用 — TaskWeaver + Djinn Stealer 双料武器

CVE-2026-48558 CVSS 10.0 · CISA KEV 7/2 截止

6月30日 The Hacker News 报道，未知威胁组织已利用 SimpleHelp 远程监控管理（RMM）软件的 CVSS 10.0 满分认证绕过漏洞 CVE-2026-48558，向受害者投送两套前所未见的恶意软件 TaskWeaver（Node.js 加载器）和 Djinn Stealer（跨平台信息窃取器）。CISA 于 6/29 将其新增至 KEV 目录，FCEB 机构修复截止日为 7/2。

漏洞机制/攻击方式：

  • 类型：OIDC 认证绕过 → 任意 Technician 账户创建（CWE-347 加密签名验证不当）
  • 攻击方式：未认证攻击者伪造 OIDC 令牌，在 RMM 服务器上创建任意 Technician 账户，绕过 MFA（首次登录可自注册 MFA）
  • 影响范围：SimpleHelp 5.5.15 及更早 / 6.0 预发布，全球数千家 MSP + 客户终端
  • 后果：攻击者获得完全 Technician 权限，可远程管理所有托管终端、执行脚本、传输文件、部署 TaskWeaver + Djinn Stealer

值得关注的原因：

  • CVSS 10.0 满分 + CISA KEV 已收录 + 7/2 截止——三重最高优先级
  • 攻击者滥用合法 Microsoft Dev Tunnels 通道（a.dev-tunnels.com），多数安全设备不会告警
  • 同时窃取 AI 开发助手凭据（Claude / Gemini / Codex / Cline），一旦失守可被用于供应链投毒
  • MSP 链式反应：单一 RMM 沦陷 → 数千家客户终端全控
  • Djinn Stealer 跨平台支持 Windows / macOS / Linux，窃取范围覆盖云、IaC、代码、AI、加密货币
攻击者 伪造 OIDC 令牌 认证绕过 创建 Technician 部署 TaskWeaver Djinn Stealer 窃密

Oracle E-Business Suite CVE-2026-46817 在野利用确认

CVE-2026-46817 CVSS 9.8 · 蜜罐证实

Defused Cyber 披露其 Oracle EBS 蜜罐在 6/27-6/28 周末观察到主动攻击。CVE-2026-46817（CVSS 9.8）位于 Oracle Payments 组件，由不当权限管理导致未认证攻击者完全接管。

受影响组件：

组件风险
Oracle Payments 12.2.3-12.2.15高危
ERP 财务系统高危
HR / 供应链 / 采购横向

关键观察：无公开 PoC，攻击者具备原始漏洞研究能力。周末攻击窗口 + Java XMLDecoder 内部执行 — SOC 几乎不可见。

3 高危

AirDrop + Quick Share 50亿设备跨平台漏洞

跨厂商协议 5B+ 设备受影响

6/30 披露，Apple AirDrop、Google Quick Share、Samsung Quick Share 三种主流文件共享协议同时暴露多个漏洞，邻近攻击者可执行内容拦截、内容篡改、身份伪造。

覆盖范围：

  • Apple 设备全系（iOS / macOS）
  • Android 设备 8 亿+
  • Samsung Galaxy 系列

三星兼容性回退可被用于冒充 Apple 设备发送伪造文件，BYOD 场景尤其危险。

4 严重

Gitea act_runner CVE-2026-58053 容器逃逸

CVE-2026-58053 CVSS 9.4 · PoC 公开

Gitea 官方 CI/CD Runner 组件 act_runner 存在容器逃逸漏洞，公开 PoC 流通中。即使配置 privileged: false，通过精心构造的 container.options 仍可注入特权能力（如 CapAdd、SecurityOpt）。

影响：

  • CI/CD Runner 特权运行 → 容器逃逸即宿主/K8s 节点权限
  • 与 exploitarium 仓库 Gitea CVE-2026-20896 形成双重攻击链
  • 自建 Gitea 实例广泛存在于亚洲中小企业研发团队

缓解：升级 act_runner ≥ 0.2.10，删除 CAP_SYS_ADMIN 等特权。

5 严重

Microsoft Defender "BlueHammer" CVE-2026-33825 仍在野利用

CVE-2026-33825 4/14 修复 · 仍在野 · 勒索软件使用

BlueHammer 本地提权漏洞利用 Windows Defender 签名更新工作流的 TOCTOU 竞争条件和路径混淆缺陷，使本地低权限用户在完全打补丁的系统上提权至 NT AUTHORITY\SYSTEM。

讽刺性：用于保护系统的杀毒软件反而成为提权途径。78 天仍在被勒索软件团伙利用。

同系列关联：

  • CVE-2026-47281 RoguePlanet（无补丁）
  • CVE-2026-50656 RoguePlanet 第二变种（PoC 公开）
