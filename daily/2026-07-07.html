1 严重

fast-mcp-telegram CVE-2026-52830 — 路径遍历把 Telegram 账户"白送"出去

7/7 当天披露 / OpenCVE+Mondoo+GHSA 三源共识 未认证 / 远程

7/7 当天公开的 CVE-2026-52830 是 fast-mcp-telegram MCP 服务器的**路径遍历漏洞**——HTTP Bearer token 被直接拼到 session 文件路径上做"会话名"用。攻击者用 `../fast-mcp-telegram/telegram` 这类 token 就能越过校验、未授权拿到默认 session 并**通过 MTProto API 操控 Telegram 账户**。OpenCVE / Mondoo / GitHub Advisory 当日同步收录，标志 MCP server 自身的权限模型成为 AI Agent 安全新焦点。

漏洞机制/攻击方式：

  • 类型：路径遍历（CWE-22）+ 认证绕过
  • 攻击方式：HTTP Bearer token 被当作路径片段，字符串 "telegram" 被黑名单拦截但未规范化输入，未限制 "../" / "/" / 绝对路径
  • 影响范围：所有 < 0.19.1 版本（0.19.1 已修复）
  • 后果：未认证远程拿到默认 session，**MTProto API 全开**（读消息 / 发消息 / 调 API / 拿附件）

值得关注的原因：

  • "token 当文件名"是经典反模式：HTTP Bearer token 应是不透明标识符而非路径片段
  • **未认证即可触发**——普通互联网节点远程即可发起
  • AI Agent 工具链 = 新攻击面：MCP server 自身的权限模型 + 文件系统访问成为焦点
  • 3 个独立数据库同日收录 = 真漏洞而非扫描器噪音
  • 修复版 0.19.1 已发，但**未自动升级的部署大量存在**
攻击者发起请求 Bearer: ../fast-mcp-telegram/telegram 路径拼接待校验 路径遍历绕过 MTProto API 操控

4 高危

TrojPix — 空隙系统 208 米 EM 渗漏通道，8.1 Mbps / 99% 准确率

山东大学 + 泉城实验室 / USENIX 2026 air-gap 数据渗漏

山东大学 + 泉城实验室披露的 **TrojPix** 通过**像素级操控**（蓝色通道最低位）让 HDMI 等数字视频接口的 TMDS 编码发出**确定性电磁辐射**——攻击者用 USRP X310 商业无线电设备**208 米外**接收（穿透 30cm 混凝土墙），峰值 **8.1 Mbps**（既往 EM 通道的 27 倍），99% 准确率，100MB 文件 < 2 分钟。

关键点：无需管理员权限，全程用户态；但必须先植入（USB / 供应链 / 内部威胁）

速览区

6. OpenSSH 10.4 — 多安全修复 + 实验性 ML-DSA+Ed25519 后量子组合签名

修复 sftp / scp 越界写 / sshd internal-sftp 静默截断 / GSSAPI 预认证 DoS / 客户端 UAF；新增 ML-DSA 44+Ed25519 组合签名（需显式开启）；Linux seccomp 沙箱失败变 fatal。

7. Microsoft Edge CVE-2026-57992 — Chromium UAF RCE，CVSS 7.5

UAF 内存破坏 + 两次点击触发 autofill；影响 Edge 150.0.4078.48（基于 Chromium 150.0.7871.47），7/3 发布；无补丁无 PoC；缓解：监控 MSRC + 启用 Enhanced Security Mode + 临时禁用 autofill。

8. Gemini Live Voice Session Flaw — token misconfig → 提示词注入 + 代码执行

BidiGenerateContentConstrained 端点临时 token 铸造时未传 live_connect_constraints → 客户端 setup 帧未锁定；攻击者覆盖系统提示词 + 启用 codeExecution → 在 gVisor 沙箱内真实执行 Python。

9. Microsoft Device Code 钓鱼 — Securelist 4-5 月大规模活动

钓鱼邮件（伪装律所通知 + 密码保护 PDF） → 假门户（带 CAPTCHA） → 一次性 code → 受害者去真实的 microsoft.com/devicelogin 输入 → 攻击者拿到 access token → 读邮件 / OneDrive / Teams。

10. SilverFox ValleyRAT — 8 阶段投递链 + Go RAT + 内核 rootkit

DLL 旁路 → 关闭日志/AV → 像素隐写（PNG）藏下一阶段 → 提权后再藏第二张图 → Donut shellcode loader → Go RAT（WebSocket + QUIC C2）→ svchost 注入 AV Killer → 内核 rootkit（65+ 命令码）。
