1 严重

Januscape CVE-2026-53359 — Linux KVM Shadow MMU 潜伏 16 年的客户机→宿主机逃逸

7/7 Hyunwoo Kim / @v4bel / Google kvmCTF 披露 kvmCTF 25 万美元奖励

7/7 公开的 CVE-2026-53359 是一项代号 "Januscape" 的 KVM Shadow MMU Use-After-Free，自 2010 年 Linux 内核 2.6.36 引入（commit 2032a93d66fa）起在 x86 影子 MMU 代码里潜伏约 16 年才被 Google kvmCTF 研究员 Hyunwoo Kim（@v4bel）发现；当前 PoC 至少能让恶意客户机让宿主机崩溃（DoS），研究员声称手握可实现宿主机代码执行的未公开利用——所有云厂商 KVM 实例都应被视为受影响。

漏洞机制/攻击方式：

  • 类型：Use-After-Free（CWE-401），影子 MMU 代码缺陷
  • 攻击方式：影子页表 EPT/NPT 处理未正确同步的引用 → mmio/page fault/tlb flush 触发 UAF → 客户机 root 在主机 ring 0 执行
  • 影响范围：Linux KVM x86 影子 MMU（Intel + AMD 全平台）
  • 后果：宿主机 ring 0 代码执行 → 访问同机所有客户机内存/磁盘

值得关注的原因：

  • "16 年才被发现"是历史级别 → AWS/GCP/阿里云/腾讯云/Oracle/华为云 几乎全部使用
  • 宿主机逃逸 = 多租户云的最坏情况：单一客户机沦陷影响同机所有客户
  • 研究员声称已有完整利用：公共 PoC 虽只是 DoS，但完整 RCE 可能已在地下流传
  • 影子 MMU 修复具有连锁影响：补丁可能影响虚拟化性能 → 部署前需回归测试
  • 云厂商 KVM 实例优先级最高：IaaS 提供商需紧急评估并推送补丁
客户机 root 页表切换 + mmio 影子 MMU UAF 宿主机 ring 0 执行 同机全部沦陷
2 严重

Adobe ColdFusion CVE-2026-48282 — CVSS 10.0 满分路径遍历，已被积极利用

Adobe 官方 / KEVIntel / SecurityAffairs 多源 CVSS 10.0 满分

CVE-2026-48282 是 CVSS 10.0 满分路径遍历，影响 ColdFusion 2025.9 / 2023.20 及更早版本，正在被积极利用，Scope Changed、无需用户交互、无需认证。RDS 服务处理文件操作时未充分校验 ../ 等相对路径片段 → 攻击者写 WebShell → 远程代码执行。

修复：升级至 ColdFusion 2025.10+ / 2023.21+

关键点：CVSS 10.0 + 在野利用 → CISA KEV 收录只是时间问题 → 必须当日修复

UNK_MassTraction — 中国关联组织利用 Roundcube CVE-2024-42009 攻击美国大学物理系

Proofpoint / 7/7 披露 CVSS 9.3 XSS

UNK_MassTraction（Proofpoint 命名，中国关联集群）利用 Roundcube CVE-2024-42009（CVSS 9.3 XSS，`message_body()` 反消毒缺陷）攻击美国 / 加拿大物理与工程院系（国家安全 / STEM 研究关联）。

攻击链：钓鱼邮件 → Roundcube XSS → 凭据 + 2FA token 窃取 → 创建邮件转发规则 → IceCube 加载器 → VShell 远控 → 长期驻留

关键点：中国关联 + STEM 目标 = 国家级 APT 行为；高校 Roundcube 部署极广 → 攻击面大

5 高危

伊朗 Cavern C2 框架 — 模块化 C2 攻击以色列 IT/政府，与 MuddyWater / Lyceum 重叠

Check Point Research / 7/6 国家级 APT

Cavern Manticore（CPR 命名）自 2026 年初持续活跃，模块化 C2 "Cavern C2" / "Cav3rn" 支持 HTTP/DNS/WebSocket + 多种 payload；针对以色列 IT 提供商 → 借助客户访问权限进入政府机构。与 MuddyWater / Lyceum 工具与基础设施重叠。

关键点：中东网络战 2026 持续升级；模块化 C2 = 长期基础设施，单一防御无法根除；IT 提供商是供应链跳板

6. ModSecurity CVE-2026-52747 + CVE-2026-52761 — WAF 解析器差异绕过，3.0.16 已修复

multipart/form-data 解析器静默剥离换行符造成解析器差异；`t:utf8toUnicode` sizeof() 错误使用（仅 32 位 i386）；影响 Apache / NGINX / IIS；3.0.16 已发布。

7. Gitea Docker CVE-2026-20896 — 通配符反向代理信任 13 天后被探测

CVSS 9.8；`app.ini` 模板硬编码 `REVERSE_PROXY_TRUSTED_PROXIES = *` → 任意来源 IP 通过 `X-WEBAUTH-USER` 伪造管理员；影响 ≤ 1.26.2；Sysdig 报告第 13 天已探测；6,000+ 实例暴露。

8. PamStealer — macOS 信息窃取器，利用 PAM API 验证密码

Jamf Threat Labs 7/2 披露；仅 Apple Silicon Mac；仿冒 Maccy → 虚假网站 → 恶意 AppleScript；PAM API 本地验证登录密码 → 滤掉无效密码；C2 avenger-sync[.]live 等。

9. FortiBleed 升级 — 430K Fortinet 设备目标，INC / Lynx 勒索团伙深度参与

披露日 2026/6/17；430,000+ Fortinet 防火墙被定向，409 管理员访问确认，354 完整攻击链执行；INC Ransom / Lynx 关联；重大受害者 FoxConn / Samsung / Oracle。

10. Microsoft SharePoint CVE-2026-45659 — KEV 截止 +3 天仍逾期，活跃攻击持续
