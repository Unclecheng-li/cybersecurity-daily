exploitarium 仓库 — libssh2 预认证 RCE + Gitea 认证绕过

130+ PoC | 匿名研究者「bikini」 CVSS 9.2 / 9.4

6 月 23 日匿名 GitHub 账户 bikini 公开聚合仓库，一次性放出 22 个软件项目共 130+ 个 PoC，且全部未提前通知厂商。6 月 28 日登顶 Hacker News 第 5 名 726 分；6 月 29 日安全社区确认其中两枚为高危可武器化零日。

漏洞机制/攻击方式：

  • CVE-2026-55200 (libssh2 ≤ 1.11.1)：ssh2_transport_read() 未上界校验 packet_length，整数溢出 → 堆越界写入 → 预认证 RCE
  • CVE-2026-20896 (Gitea ≤ 1.26.2 Docker)：REVERSE_PROXY_TRUSTED_PROXIES 默认通配符 *，任意源 IP 发送 X-WEBAUTH-USER: admin 即可获得管理员权限
  • 触发条件：客户端连接恶意 SSH 服务 / 任意源 IP 一行 HTTP Header
  • 影响范围：curl、Git、PHP、MTA、自建 Gitea 全栈中招
  • 后果：预认证 RCE + 管理员接管，构成 CI/CD 全链路攻击

值得关注的原因：

  • libssh2 是 curl/Git/PHP 的传递性依赖，爆炸半径极广
  • Gitea Docker 镜像默认通配符，一行 Header 即可接管
  • Femtosec 报告攻击者已开始重新包装公共 PoC 工具
  • 与 Chaotic Eclipse 4-5 月未协调披露事件形成同一研究生态
攻击者 恶意 SSH 握手 packet_length 溢出 预认证 RCE 客户端沦陷

Gitea act_runner 容器逃逸 CVE-2026-58053

CI/CD 工具链 | 公开 PoC CVSS 9.9

Gitea 官方 CI/CD Runner 组件 act_runner 存在容器逃逸漏洞，攻击者控制 Runner 上的恶意 workflow 后可逃逸至宿主机，获得 Kubernetes 节点权限。

关键属性：

  • 类型：容器逃逸 → 宿主机 RCE
  • 版本：act_runner < 0.2.10
  • PoC 状态：已公开
  • 威胁：直接接管企业内网代码发布通道

Defender RoguePlanet 零日 CVE-2026-47281

Windows 10/11 | 在野利用 CWE-20 / LPE

Microsoft Defender 实时扫描引擎中的输入验证不当漏洞，研究人员 Chaotic Eclipse 6/10 公开 PoC，已确认在野利用。已安装 6 月补丁的系统仍可被利用。

关键属性：

  • 类型：CWE-20 输入验证不当 → LPE → SYSTEM
  • 触发：VS Code 入口触发 Defender 扫描逻辑缺陷
  • 状态：在野利用 / 补丁研发中
  • 影响：Windows 11 / Windows 10 全版本

速览区

6. Windows Secure Boot 证书过期持续发酵

Microsoft UEFI KEK CA 2011 (6/24) 与 UEFI CA 2011 (6/27) 相继失效，未更新 UEFI 固件的系统出现启动失败，涉及全球数十亿台 Windows / Linux 设备。

7. SimpleHelp 认证绕过 CVE-2026-48558 入 KEV

CISA 6/28 将 SimpleHelp 远程支持软件认证绕过漏洞新增至 KEV，BOD 26-04 3 天期限适用；MSP 通过该漏洞可横向控制多个客户终端。

8. cURL 8.21.0 一周后回顾 — 25 年老漏洞

cURL 8.21.0 修复 18 个 CVE，含潜伏 25 年的 URL 解析逻辑漏洞 CVE-2026-8932；Windows / macOS 终端用户需手动升级。

9. AI 框架漏洞持续 — Mastra 供应链后续影响

Mastra 供应链投毒（朝鲜关联）持续监测；LiteLLM CVE-2026-42271（逾期 7 天）、Flowise CVE-2026-56274、Splunk AI Toolkit CVE-2026-20266（在野利用）均未完全处置。
