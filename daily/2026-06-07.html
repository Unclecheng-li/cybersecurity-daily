1 严重

Cisco SD-WAN 零日漏洞 CVE-2026-20245（第7个零日，无补丁）

CVE-2026-20245 | 影响所有部署类型 在野利用 · 零日无补丁

思科披露了影响 Catalyst SD-WAN Manager 的第7个零日漏洞 CVE-2026-20245。该漏洞允许经过身份验证的攻击者通过上传恶意文件以 root 权限执行任意命令。尽管思科尚未发布补丁，但已确认该漏洞正在被积极利用。

漏洞机制/攻击方式：

  • 类型：特权提升（命令注入）
  • 攻击方式：CLI命令处理模块未对文件内容进行充分验证，攻击者上传恶意文件触发命令注入
  • 影响范围：所有部署类型（本地、Cloud-Pro、FedRAMP）
  • 后果：以root权限执行任意命令，向边缘设备推送配置更改

值得关注的原因：

  • 攻击链组合：该漏洞很可能与 CVE-2026-20182（认证绕过）和 CVE-2026-20127 组合利用，形成完整的攻击链
  • 无补丁无缓解：思科既未发布补丁，也未提供临时解决方案
  • 2026年第7个零日：这是思科SD-WAN产品在2026年遭遇的第7个零日漏洞，表明该产品线已成为高级威胁行为者的重点目标
  • 发现方：Mandiant（谷歌旗下）
攻击者 获取netadmin权限 上传恶意文件 命令注入 Root权限执行 边缘设备沦陷
2 严重

Chrome 149 修复创纪录的 429 个安全漏洞

CVE-2026-10881 (CVSS 9.6) | 22个关键漏洞 已修复

谷歌发布了 Chrome 149 稳定版更新，一次性修复了创纪录的 429 个安全漏洞。这是 Chrome 历史上单次更新修复漏洞数量最多的一次，其中包括 22 个关键级别的漏洞。

受影响组件：

组件漏洞数风险
Angle（WebGL库）37高危
扩展接口18高危
媒体处理28中高危
其他组件346中危

值得关注的原因：

  • AI辅助漏洞发现：谷歌推测专用AI工具（如Google Big Sleep）在发现漏洞数量激增中扮演重要角色
  • 高额赏金：谷歌为此支付了约20.9万美元的漏洞赏金
  • 漏洞类型集中：释放后使用（UAF）漏洞110个，占比25.6%
3 高危

Polyfill.io 供应链攻击持续发酵（东芝、无印良品受影响）

2024年被恶意接管的CDN服务 持续影响中

2024年被恶意接管的 Polyfill.io CDN 服务继续影响全球网站。东芝和无印良品等知名品牌网站上出现了可疑的登录弹窗，试图窃取用户凭据。攻击方式是通过返回 HTTP 401 状态码触发浏览器内置登录弹窗进行钓鱼。

受影响品牌：

  • 东芝（Toshiba）
  • 无印良品（Muji）
  • Zojirushi
  • 三星智能电视（Samsung Smart TV）

值得关注的原因：

  • 持续影响：该供应链攻击始于2024年，至今仍在影响新网站
  • 攻击手法隐蔽：利用HTTP 401状态码触发浏览器内置登录弹窗，难以被用户识别
  • 技术债务风险：众多网站因遗留代码中包含旧的Polyfill.io引用而受影响
4 严重

UNC5221（VerdantBamboo）APT 组织使用新恶意软件持续攻击

Brickstorm + Plenet + AgentPSD 持续活跃

被追踪为 UNC5221（又名 VerdantBamboo）的中国间谍组织被发现使用 Brickstorm 后门以及此前未记录的 Plenet 和 AgentPSD 恶意软件来访问 Microsoft 365 环境。该组织自 2023 年以来一直活跃，擅长利用边缘设备零日漏洞。

攻击工具集：

工具名称类型开发语言
Brickstorm后门Golang → Rust
Plenet后门.NET
AgentPSD反向ShellPython

值得关注的原因：

  • 多语言恶意软件：使用Go、Rust、.NET、Python四种语言开发工具
  • 长期潜伏：在受害者网络中潜伏超过18个月才被发现
  • 供应链攻击：通过入侵托管服务提供商（MSP）扩散到下游客户
5 高危

五眼联盟警告：中国间谍利用虚假招聘平台招募西方人员

LinkedIn、Indeed、Upwork 持续进行中

五眼联盟（FBI、MI5、澳大利亚、加拿大和新西兰情报机构）发布联合公告，警告中国情报机构正在通过 LinkedIn、Indeed、Upwork 等专业招聘平台，以虚假招聘为掩护，对西方人员进行有针对性的招募。

目标人群：

  • 持有安全许可的人员
  • 军事人员
  • 政府雇员
  • 国防、外交、情报及安全相关领域的从业者

值得关注的原因：

  • 攻击企业化：招募流程设计专业，难以被察觉
  • 非机密信息也危险：即使是非机密的政府政策、军事战略信息，也可与其他敏感报告结合形成情报拼图
  • 支付方式多样：包括PayPal、Wise、加密货币等多种支付方式
