Node.js CVE-2026-48930 — TLS 证书授权机构静默重绑定，CVSS 9.8 Critical

嵌入空字节主机名导致 TLS 连接静默劫持 | 影响 Node.js 22.x/24.x | 120万+站点 | C层/JS层处理不一致

1 严重

Node.js CVE-2026-48930 — TLS 证书授权机构静默重绑定

CVE-2026-48930 | Node.js 22.x/24.x | CVSS 9.8 6月安全发布

Node.js 2026年6月安全发布修复了其历史上最严重的 TLS 漏洞之一。CVE-2026-48930 允许攻击者通过在主机名中嵌入空字节（\0），利用 C 语言解析器与 JavaScript 验证器之间的处理差异，实现静默 TLS 证书授权机构重绑定。

漏洞机制：

  • 类型：CWE-170 空字节截断 → TLS 证书授权重绑定
  • 攻击方式：嵌入空字节的主机名被 C 层 getaddrinfo() 截断，但 JS 层验证器不截断
  • 影响：HTTPS/WSS 连接被静默重定向到攻击者服务器，客户端无告警
  • 影响范围：Node.js 22.x 和 24.x 所有受支持版本，120万+站点

值得关注的原因：这是 TLS 客户端层面的设计缺陷，影响所有使用 Node.js 的 API 网关、微服务和 SSR 应用。同批次还修复了 CVE-2026-48618（通配符绕过）、CVE-2026-50741（修复绕过）等共 12 个漏洞。

PTC Windchill CVE-2026-12569 — 未授权 RCE + Webshell

CVE-2026-12569 | CVSS 9.3 | CISA KEV 6/28截止 在野利用

PTC Windchill PDMlink 和 FlexPLM 曝出未授权远程代码执行漏洞，由于反序列化不受信任数据，攻击者可执行任意代码。PTC 确认攻击者正在部署 JSP Webshell，CISA 于 6/25 加入 KEV 目录。

PTC 产品首次进入 CISA KEV。制造业（航空航天/汽车/国防）广泛使用，沦陷意味着产品设计图纸和供应链数据暴露。

3 严重

Amazon Q CVE-2026-12957 — MCP 自动执行窃取云凭证

CVE-2026-12957 | Wiz Research | 已修复 v1.65.0 高危

Wiz Research 发现 Amazon Q Developer VS Code 扩展在加载工作区时自动执行 .amazonq/mcp.json 中的 MCP 配置，无需用户同意。攻击者创建恶意仓库，开发者克隆打开后 AWS 凭证即被窃取。

跨生态影响：Claude Code、Cursor、Windsurf 均存在类似问题。MCP 自动执行已成为 AI 开发工具的行业级安全隐患。

4 高危

DirtyClone CVE-2026-43503 — Linux 内核本地提权

CVE-2026-43503 | CVSS 8.8 | JFrog 发现 DirtyFrag家族第5个

Linux 内核 XFRM/IPsec 子系统本地提权漏洞。通过克隆网络数据包丢失安全标志，覆写页面缓存中的 /usr/bin/su 认证逻辑获取 root。磁盘文件不变，对文件完整性监控和审计日志完全不可见。

影响：Debian/Fedora 默认受影响，云环境和容器多租户场景风险最高。DirtyFrag 家族已累计 5 个变种。

5 高危

0DIN 演示 — 干净 GitHub 仓库劫持 AI 编码代理

6月25日 | Mozilla 0DIN | 无 CVE 概念验证

Mozilla 0DIN 演示：一个完全干净的 GitHub 仓库可让 Claude Code 打开反向 Shell。恶意 payload 不存在于仓库中，而是通过 DNS TXT 记录在运行时获取。

"Claude Code 从未决定打开一个 shell。它决定修复一个错误。" — 三层间接引用使攻击对所有扫描器不可见。

6 严重

Cisco UCM CVE-2026-20230 — SSRF → Root 提权

CVSS 9.8 | CISA KEV 6/28截止 在野利用

Cisco Unified Communications Manager Web 组件 SSRF 漏洞，攻击者可通过未授权访问实现任意文件写入并提权至 Root。CUCM 是企业/政府电话系统核心，沦陷后通话录音和 SIP 信令全部泄露。

PoC 已于 6/24 公开，今日是 CISA KEV 修复截止日。

7 高危

DCloud Uni-App 被滥用 — 20万投资诈骗网站

Infoblox 报告 | 6月27日 诈骗基础设施

威胁行为者利用中国开源跨平台框架 DCloud Uni-App 批量创建投资诈骗网站模板。Infoblox 已识别超过 200,000 个诈骗网站，主要伪装为加密货币/股票投资平台。

这不是框架漏洞，而是合法开源工具被大规模武器化的典型案例。
