GitLab 同日修复 6 个 CVE

5月27日 NVD批量发布 覆盖 GitLab EE 17.1～19.0 版本

NVD 5月27日批量发布 GitLab 安全公告，涉及6个CVE，覆盖 GitLab EE 17.1～19.0 版本，建议所有自托管 GitLab 实例立即检查并升级。

  • 在多租户场景下尤其危险，建议批量升级

Nx Console 恶意版本惊魂 18 分钟（CVE-2026-48027）

CVSS 9.8 供应链投毒 5月27日确认

Nx Console（用于 Angular/Nx 开发工具链的 VS Code 扩展）在 Visual Studio Marketplace 遭遇供应链投毒：

5月19日 12:30 UTC 恶意版本 18.95.0 被发布 12:48 UTC 约18分钟后被移除

  • 这是本月第二起 VS Marketplace 供应链投毒事件
  • 开发者工具的供应链投毒可直接污染企业 CI/CD 流水线
  • 影响范围远超普通终端用户
  • 建议：检查是否安装了 18.95.0 版本，撤销相关 GitHub Action 工作流 tokens
3 严重

GitBucket CVE-2018-25332（CVSS 9.8）重新标注

CVSS 9.8 老漏洞新标注 5月17日 NVD 收录

GitBucket 4.23.1 版本存在未经认证的远程代码执行漏洞。CVE-2018-25332 是2018年的老漏洞，但近期被 NVD 重新收录并标注为高危。

  • 2018年的老漏洞被重新标注为高危

  • 如果此前未修补，攻击者可无需任何凭证即可获得服务器权限
  • 建议：检查版本并立即升级
4 严重

NVIDIA Triton Inference Server 认证绕过漏洞（CVE-2026-24207）

CVSS 9.8 AI基础设施 5月20日 NVD 收录

NVIDIA Triton Inference Server 是一款开源 AI 模型推理服务组件，广泛部署于 Linux 生产环境。漏洞存在于 Vertex AI 兼容入口 /predict，该入口在特定条件下绕过认证检查。

项目详情
CVECVE-2026-24207
类型认证绕过
CVSS9.8（Critical）
影响代码执行（通过认证绕过）
受影响组件Vertex AI 兼容入口 /predict

值得关注的原因：

  • CVSS 9.8 — 接近满分，已是最高等级威胁
  • Triton Inference Server 广泛部署 — 几乎所有使用 AI 模型的生产环境都可能受影响
  • /predict 端点 — 是最常用的推理接口，攻击面直接暴露于互联网
  • AI 推理服务器常含敏感数据 — 模型权重、训练数据、用户推理请求均可被窃取或篡改
