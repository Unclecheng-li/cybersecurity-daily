Magento RCE零日漏洞被CISA收录KEV：单个Cookie即可接管电商服务器

CVE-2026-45247 CVSS 9.8，Mirasvit Cache Warmer插件PHP反序列化缺陷，一周内从披露到收录创纪录

1 严重

Magento RCE零日 CVE-2026-45247 被CISA收录KEV

CVSS 9.8 在野利用

CISA于6月3日正式将CVE-2026-45247添加至已知被利用漏洞（KEV）目录。该漏洞存在于Magento电商平台广泛使用的Mirasvit Cache Warmer缓存插件中。

漏洞机制：

  • 类型：PHP反序列化不受信任数据
  • 攻击方式：未认证攻击者通过构造恶意Cookie即可触发
  • 影响范围：任何Magento商店前台页面
  • 后果：远程代码执行（RCE），完全接管服务器

值得关注的原因：

  • Sansec安全公司于5月26日首次披露，一周内即被CISA收录
  • Magento全球电商市场份额巨大，涉及海量在线商店
  • 单个恶意Cookie即可完成攻击，门槛极低
攻击者 恶意Cookie Magento前台 PHP反序列化 RCE执行 服务器接管

3 高危

Microsoft 365 Android FlagLeft：调试标志遗留致数十亿用户令牌泄露

CVE-2026-41100/41101/41102/42832 已修复

研究人员发现多个Microsoft 365 Android应用中遗留了一个开发调试标志setIsDebugMode(true)，导致应用间身份验证检查被绕过。

受影响应用（累计下载数十亿次）：

  • Microsoft Word
  • Microsoft PowerPoint
  • Microsoft Excel
  • Microsoft 365 Copilot
  • Microsoft Loop
  • OneNote
4 中高危

Windows Search URI：点击链接即泄露NTLMv2哈希

状态：未修补

研究人员披露了一个存在于Windows search: URI处理程序中的未修补漏洞。攻击者诱导用户点击特制链接后，系统会自动连接到攻击者控制的SMB服务器。

恶意链接 用户点击 SMB连接 哈希窃取
5 中高危

Google Gemini通知劫持：WhatsApp/Slack消息变身攻击向量

状态：已修补

研究人员发现，通过WhatsApp、Slack、SMS、Signal等应用发送的一条恶意通知，即可劫持Android设备上的Google Gemini语音助手。

攻击能力：

  • 打开任意窗口
  • 伪造消息
  • 加入通话
  • 污染Gemini长期记忆
